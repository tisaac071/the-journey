# the-journey

This repository contains a lightweight, responsive one-page portfolio for "Isaac Tumusiime" with an updated classic-modern interface.

Preview locally:

```bash
# from the repository root
python3 -m http.server 8000 --directory .
# then open http://127.0.0.1:8000 in your browser
```

Files added:

- `index.html` — main one-page portfolio
- `styles.css` — site styles and responsive layout
- `script.js` — small UI helpers (mobile nav, smooth scroll)
